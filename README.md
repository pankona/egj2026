# Rift

[Ebitengine Game Jam 2026](https://itch.io/jam/ebitengine-game-jam-2026) 参加作品。テーマは **DISCONNECT**。

ドラッグ開始点から方向へ**光の衝撃波**がズバッと伸びて闇を切る。ドラッグ距離で線の長さが 3 段階に変わり、エネルギーを 1〜3 ユニット消費。3 連発の直線でポリゴン状に囲んだ暗領域は一気に光で塗りつぶされる (claim)。敵は光の縁を侵食して取り戻しに来るし、複数体集まると敵側も暗線で囲い返してくる。**ステージの敵を全滅させるとクリア**。10 ステージ目は巨大ボスを光で封じる。

## 操作

- **ドラッグ** (マウス／タッチ): ドラッグ開始点が衝撃波の起点。方向と距離で線の長さが決まる
  - 短く引く (〜100px): 短い衝撃波 = 1 ユニット消費
  - 中くらい (〜200px): 中くらいの衝撃波 = 2 ユニット消費
  - 長く引く (200px〜): 長い衝撃波 = 3 ユニット消費
- バッファは **6 ユニットまでチャージ可能**、毎秒 1 ユニットずつ自動回復。3 unit 大ぶり 2 連発も、1 unit 小撃ち 6 連射も両立する
- ドラッグ中はリリース前に「いま離すと何が出るか」がプレビュー表示される (線の太さ・透明度・起点リングの大きさが unit 数に応じて変わる)
- 斬撃で閉じ込めた暗領域は自動で光化 (claim)。内部の敵もろとも消える
- タイトル・リトライ・全クリア画面では **クリック / Space**

## 実行

ブラウザ (WebAssembly):

```
make serve-wasm
# → http://localhost:18081/
```

ファイル保存ごとに wasm を再ビルド + devserver を再起動するホットリロードは air を使う (初回のみ `make install-tools`):

```
make watch
# → http://localhost:18081/  保存で自動再ビルド、ブラウザは F5
```

ネイティブ実行は `make run` で可能だが、配布形態は wasm に絞っている。

他のターゲットは `Makefile` 参照。デバッグ表示 (FPS) は `DEBUG=1 make run` または `http://localhost:18081/?debug=1`。
